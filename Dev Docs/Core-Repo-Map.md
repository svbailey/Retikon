# Core Repo Map (OSS)

Core is the open-source, cloud-agnostic runtime. It must not depend on any
cloud provider SDKs.

- `retikon_core/`
- `local_adapter/`
- `retikon_cli/`
- `sdk/`
- `frontend/dev-console/` (Core UI)
- `tests/core/`
